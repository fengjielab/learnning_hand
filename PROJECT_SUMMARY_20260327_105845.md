## ✅ 已完成配置

### 1. VirtualBox USB 配置
- 安装 VirtualBox Extension Pack
- 配置 USB 3.0 控制器
- 添加 Omega.7 到 USB 过滤器
- 设备 ID: `1451:0402`

### 2. udev 权限配置
- 创建规则文件：`/etc/udev/rules.d/40-haptic-device-udev.rules`
- 允许普通用户访问 USB 设备

### 3. 依赖安装
- SDK 版本：3.17.6
- Python 虚拟环境：handvenv (uv 管理)
- 已安装：forcedimension-core, mujoco, scipy, numpy

## 🚀 快速开始

### 运行前释放设备
```bash
cd teletest_forcedimension
uv run python3 utils_20260327_104135/device_manager.py full
```

### 测试程序
```bash
# 基础连接测试
uv run python3 test_omega7_debug_20260327_103400.py

# 力反馈测试（5 种模式）
uv run python3 test_force_feedback_20260327_103352.py

# Mujoco 机械臂控制（需要桌面环境）
uv run python3 test_mujoco_20260327_103622.py
```

---

## ⚠️ 注意事项

1. **停止程序按 Ctrl+C**，不要按 Ctrl+Z（会挂起占用设备）
2. **Mujoco 测试需要桌面环境**，不能在 SSH 终端运行
3. **运行新程序前先释放设备**：`uv run python3 utils_20260327_104135/device_manager.py full`
